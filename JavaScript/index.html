基本文法

  • 変数

    • javascriptでは変数を扱うことができます。変数を指定するときはvarとletを用いて変数を指定します。
    もちろん変数なので変数の中の値はプログラム内で変更することができます。今回はとりあえずvarを使っていきます。そもそものこの二つのの変数の違いはなんなのかについてはのちおほど説明しようと思います。
    letはいろいろめんどくさい。慣れればこれはすごい機能!と言うかbarよりletの方が推奨されるらしい...

  • 定数

    • 変数とは違いこの定数は一度設定したらプログラム内で変更することはできません。
    文字通り定められた不変の値というわけです。
    ちなみに変数はJavaScriptではconstを用いて指定することができます

  • 値の出力

    • JavaScriptでは変数や定数、配列などに様々な情報やプログラムの発火装置を埋め込みます。
    それらを呼び出すときにはいくつか方法があります。最も簡単で重要なものがconsole.logです。
    詳しい説明は口頭で行います。

  • オブジェクト

    • オブジェクトとは簡単にいうとプログラム部品です。このオブジェクトをいくつも組み合わせることで一つのプロダクトを作ります。
    オブジェクトには3種類のものが存在する、

    1. 組み込みオブジェクト

      2. JavaScriptであらかじめ定義されているもの

    2. ブラウザオブジェクト

      3. ブラウザで取り扱えるもの

    3. DOMオブジェクト

      4. HTMLの要素のアクセスできるもの

      ちなみにこれはいずれ話すかもしれませんがJSXという書き方が存在します。通常JSXは.jsxファイルの記述して行きます。
      しかし、これをES6では.jsxファイルではなく.jsファイルで記述できるようになっています

  • 関数

    • 今まで基本的な文法を学んだと思います。しかしwebアプリやネイティブアプリを作成するにあたり基本文法単体では難しいです。
    そこで複数の処理をまとめていつでもどこでも使えるようにしちゃおう！と言うのが関数です。
    少し正確に説明すると「よく利用する処理内容を一つの箱の中にしまっておき、利用したい時に箱の名前を一行記述することで実行することができる機能。」
    特定の処理を分割して作成するので何かの拍子にエラーが起きた場合対応しやすくなる。
